# nouvil-academy-website
Description
This is a complete, responsive website for Nouvil Academy - an Arabic programming education platform. The website features course listings, problem-solving quizzes, learning tracks, and a fully Arabic RTL (right-to-left) interface.

# Features
Fully responsive design (mobile, tablet, desktop)

Arabic RTL layout and typography

Course catalog with filtering

Problem-solving quizzes section

Learning paths and tracks

Modern UI with professional styling

Fast loading and optimized performance

Technologies
HTML - Semantic markup

CSS - Styling and animations

JavaScript - Interactive functionality
